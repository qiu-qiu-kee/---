## 简介
Element UI表单设计及代码生成器，可将生成的代码直接运行在基于Element的vue项目中；也可导出JSON表单，使用配套的解析器将JSON解析成真实的表单。 
## JSON解析器
将保存在数据库中的JSON表单，解析成真实的表单  
```
// 安装
npm i form-gen-parser
```
[查看在线示例](https://mrhj.gitee.io/form-generator/#/parser) 

## 运行
- 确保已经安装node.js 10+
- 首次下载项目后，安装项目依赖：
```
yarn
```
或
```
npm install
```
- 本地运行
```
npm run dev
```
- 构建
```
npm run build
```
